Abstract

Sa se realize o aplicatie Web destinata colectionarilor de capace, dopuri si etichete specifice recipientelor folosite pentru inmagazinarea bauturilor (ne)alcoolice. Pe baza facilitatilor de cautare multi-criteriala implementate, utilizatorii autentificati vor putea crea, inventaria si partaja informatii privind artefactele detinute in functie de diverse caracteristici: atribute fizice, imagine, valoare, tara de provenienta, istoric al folosirii efective etc. Se va oferi suport pentru import/export de date. Se vor genera statistici diverse disponibile in formate deschise – minimal, CSV si DocBook. Se va realiza, de asemenea, un clasament al celor mai populare obiecte colectate, disponibil si ca flux de date RSS.

Tehnologii

Tehnologii utilizate:

  • Baze de date: MySQL
  • Prototip: HTML + CSS
  • Backend: NodeJS
  • Server: Apache
  • AJAX

Arhiterctura aplicatie

Am folosit o arhitectura de baza, adica Clientul trimite o cerere Serverului, Serverul apeleaza Baza de Date, iar raspunsul este trimis catre Client tot prin intermediul Serverului.

Descriere functionalitati

Vizitator:

  • Logare: permite logarea
  • Înregistrare: permite crearea unui cont nou
  • Vizualizare articole: permite vizitatorului să vizualizeze toate articolele publicate

User:

  • Editare profil: Userul isi poate modifica profilul
  • Vizualizare articole: poate vedea la fel ca un vizitator articolele publicate
  • Adaugare articol: permite userului postarea de articole
  • Adaugare articol in propriul cart: permite userului sa adauge articole publicate intr-un cart propriu, pe care acesta le va putea exporta in format CSV
  • Exportarea cartului: permite userului exportarea articolelor din cartul prorpiu
